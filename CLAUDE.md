## 開発コマンド
- 開発サーバー起動: `npm run dev`
- ビルド: `npm run build`
- プレビュー: `npm run preview`

## 操作方法
- **左クリック**: 艦隊を選択（円形選択枠とリングエフェクト）
- **ドラッグ選択**: 左クリック＋ドラッグで範囲選択（同盟軍のみ）
- **右クリック**: 選択中の艦隊を移動
- **自動戦闘**: 射程内の敵を自動攻撃（ビームと爆発エフェクト）
- **UI制御**: 一時停止、速度変更（0.5x/1x/2x）、音声ミュート
- **勝利条件**: 敵陣営全滅でゲーム終了
