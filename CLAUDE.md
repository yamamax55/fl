## 開発コマンド
- 開発サーバー起動: `npm run dev`
- ビルド: `npm run build`
- プレビュー: `npm run preview`

## 操作方法
- **左クリック**: 艦隊を選択（黄色枠が表示）
- **ドラッグ選択**: 左クリック＋ドラッグで範囲選択（同盟軍のみ）
- **右クリック**: 選択中の艦隊を移動
- **自動戦闘**: 射程内の敵を自動攻撃（1秒間隔）
- **HP管理**: HPバーで体力状況確認（緑→黄→赤）
