## 開発コマンド
- 開発サーバー起動: `npm run dev`
- ビルド: `npm run build`
- プレビュー: `npm run preview`

## 操作方法
- **左クリック**: 艦隊を選択（黄色枠が表示）
- **右クリック**: 選択中の艦隊を移動
- **移動**: 目標地点まで自動で滑らかに移動
