# Galaxy RTS - 銀河英雄伝説RTS

## 概要
銀河英雄伝説の世界観を基にしたリアルタイム戦略ゲーム

## セットアップ
```bash
npm install
npm run dev
```

## 操作方法
- **左クリック**: 艦隊選択（黄色枠が表示されます）
- **ドラッグ選択**: 左クリック＋ドラッグで範囲選択（同盟軍のみ操作可能）
- **右クリック**: 選択中の艦隊を指定位置に移動
- **自動戦闘**: 射程内（150px）の敵を自動攻撃（攻撃間隔1秒）
- **HP管理**: 各艦隊のHPをバーで確認（緑→黄色→赤）

- **開発環境**: Vite
- **言語**: JavaScript (ES6)

## 開発方針
1. **シンプルファースト**: 最小限の機能から始める
2. **段階的実装**: 動くものを作ってから機能追加
3. **コミット頻度**: 小さな機能ごとにGitコミット

## ブラウザでの確認方法
1. `npm run dev` でサーバー起動
2. http://localhost:3000 にアクセス
3. 画面に青と赤の四角（各3隻ずつの艦隊）が表示されることを確認
4. 左クリックで同盟軍艦隊（青）を選択し、黄色枠が表示されることを確認
5. ドラッグで複数の同盟軍艦隊を範囲選択できることを確認
6. 右クリックで選択した艦隊が移動することを確認
7. 画面上部のUIで選択艦隊数とHP情報が表示されることを確認
8. 艦隊が接近すると自動で戦闘が開始され、HPバーが変化することを確認
